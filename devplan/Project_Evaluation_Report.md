## 📋 프로젝트 개요

**목적:** 본 프로젝트는 전력 설비(스위치기어) 관련 견적 생성 및 관리 웹 애플리케이션으로, 엔지니어와 영업 사용자가 현장 정보 기반으로 신속하게 견적을 산출하고 기록할 수 있도록 설계되었습니다.

**주요 목표:**
- **목표1:** 견적 작성의 정확성과 일관성 향상
- **목표2:** 프로젝트 이력 및 변경 관리(리비전) 제공
- **목표3:** 간편한 엑셀/보고서 출력 및 공유 기능 제공

**대상 사용자:**
- **Label:** 주사용자: 전력 엔지니어, 현장 관리자
- **Label:** 보조사용자: 영업 담당자, 견적 검토자

**주요 사용 시나리오:**
- **Label:** 신규 견적 생성: 현장정보 입력 → 모듈/자재 선택 → 비용 산출 → 보고서 출력
- **Label:** 프로젝트 수정/리비전: 기존 견적 로드 → 항목 수정 → 변경 이력 저장
- **Label:** 외부 공유: 엑셀/인쇄 템플릿을 통한 보고서 배포

**전략적 위치:** 본 애플리케이션은 소규모/중견 전력 설비 업체를 위한 견적 자동화 도구로, 기존 수작업 프로세스를 대체하여 업무 효율성과 감사 가능성을 높이는 것을 목표로 합니다. SaaS 전환 또는 ERP 연동을 위한 확장 기반으로 활용 가능하며, 빠른 프로토타이핑과 커스터마이징이 장점입니다.
